- lock read access
- to unlock read access, everything must be erased
- bootloader protected by auto-write protection
- serial communication protocol with the PC at 115200 baud
	| start | start | length | length | src_id | dest_id | msg_id | payload | crc | crc | end | end |
	- the PC asks questions, the microcontroller always replies
	- response timeout: 1 second
	- inter-byte timeout: 1 second
	- acknowledgement command that acts as a capture / keep-alive
	- reset request command
	- command to write data to memory
	- command to read data from memory

- the IDs that carry data to write to memory have encrypted payloads.
- the PC software cannot decrypt them

- functions:
	- PC receive
	- PC transmit
	- protocol wrapper
	- protocol unwrapper
	- decrypt payload
	- encrypt payload
	- write to memory
	- read from memory
	- reset
